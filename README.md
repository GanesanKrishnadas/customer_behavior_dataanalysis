# customer_behavior_dataanalysis
Customer Shopping Behavior Analysis

Overview

Analyzed customer shopping behavior using Python → PostgreSQL → Power BI workflow.

Goal: uncover insights about demographics, purchasing patterns, and product categories.
________________________________________
Dataset

•	customer_shopping_behavior.csv 

•	Includes age, gender, purchase behavior, categories, review ratings, and transactions. 
________________________________________
Tools

•	Python (Pandas) – Data cleaning & EDA 

•	PostgreSQL – SQL analysis 

•	Power BI – Dashboard visualization 
________________________________________
Key Steps

1.	Load and inspect dataset 
2.	Clean data and handle missing values 
3.	Feature engineering (age groups, purchase frequency) 
4.	Exploratory Data Analysis (EDA) 
5.	Load data into PostgreSQL and run queries 
6.	Build interactive Power BI dashboard 
________________________________________
Insights

•	Identified key customer segments 
•	Highlighted high-revenue product categories 
•	Observed relationship between purchase frequency and spending
